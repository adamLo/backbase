How to use the weather app

You can add new location by touching the Add button on the top. On add locations screen you can either tap on the map or type in a city name with optionally the country code. the app will try to search for weather forecasts at given location and add to bookmarks if found one.

You can remove a bookmarked city by swiping left on it and hit "delete".

You can see detailed conditions and 5-day forecast if you tap on a bookmarked city. This also updates bookmarks.

You can search your bookmarked cities too by typing city name in the search bar.


You can change units in Settings. Note: bookmarks will be updated if you select one.
You can also clear all bookmarks in Settings.


Thanks for using my app, I hope you like it!
